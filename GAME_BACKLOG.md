# Abstract-Games Coverage Backlog

> Capability map of the abstract-game universe against this platform's primitives.
> Generated by a research workflow (228 games across 6 catalogs → ~70 distinct),
> then edited to apply the completeness-critic's corrections. The organizing
> insight: **rules logic is never a "platform gap" — a custom `Game` subclass can
> already do hoppers, custom adjacency, line-push, sowing, approach-capture on the
> existing board types. The genuine gaps are a short list of UI/renderer
> primitives.** So most "famous missing games" are actually Tier-0 work.

Shipped today (~23): tic-tac-toe, connect_four, gomoku, pente, order_and_chaos,
reversi, hex, oust, yodd, checkers, lines_of_action, amazons, breakthrough,
brandub, fox_and_hounds, foxsox, goose_chase, freeform_chess, + the ChessLike
family (chess, berolina, grand_chess, los_alamos_chess).

---

## Recommended order

1. **Drain Tier 0** (no platform change) — the biggest, cheapest win. Includes
   the entire tafl + draughts + connection + ChessLike-variant packs, the
   lightweight Go cousins, **and** (after reclassification) Xiangqi, Janggi,
   Abalone, Fanorona, Mancala. This is pure fan-out work.
2. **Build the three reusable engine cores** (code only, no UI change): a hex
   ChessLike core, a cannon/hopper core, a Go liberty/capture core. Each unlocks a
   cluster and is perft/known-result checkable.
3. **Then invest in UI primitives**, in fame-per-effort order: drops/reserve tray
   → stacking → territory-scoring → the rest.

## Tier 0 — implementable TODAY (no platform change)

Pure `Game` subclasses on existing board types (`square` / `hex` / `polygons`),
one-piece-per-cell, 2-seat, perfect-info, deterministic. Each needs an independent
correctness anchor (known result / perft / published count) before it ships.

**Tafl (square, custodial capture)** — Tablut (9×9), Hnefatafl/Copenhagen (11×11),
Tawlbwrdd, Ard Ri. Mostly sizes/rulesets; Brandub already proves the family. Some
fold into Brandub as manifest options.

**Draughts (square, jump-path)** — International (10×10, flying kings, max-capture),
Turkish/Dama (orthogonal), Frisian, Dameo, **Konane** (Hawaiian, a clean CGT
last-move-wins). Multi-jumps already encode as `>`-paths (checkers proves it).

**Connection (placement)** — Havannah (hex), Y (polygons/triangular), Trike
(polygons + one shared neutral pawn — *movement*, not placement), Crossway (square),
Unlur, ConHex. Can't-draw → automatic termination. Pie/swap already a button.

**Alignment / N-in-a-row (square)** — Connect Six (two placements/turn as a
`>`-path), Renju (Gomoku + forbidden moves), Irensei, Squore.

**ChessLike variant pack (square, ~40 lines each)** — Makruk, Shatranj, Capablanca
(10×8), Gothic, **Chess960** (randomized setup), Wildebeest (11×10), Courier
(12×8), Knightmate; plus the lichess goal-variants needing only a custom terminal:
**King of the Hill, Three-Check, Racing Kings, Horde**. (Atomic/Antichess = custom
apply/filter, still no new primitive.) Perft-checkable.

**Movement abstracts (square/hex, one-per-cell)** — **Abalone** *(critic fix:
NOT stacking — flat hex/rhombus marbles, the sumito line-push is custom move-gen;
a big-name game that's Tier-0)*, Dao (4×4), Martian Chess (zone-ownership + a
per-seat score counter — its "reserve" is a score, not a drop pool), Gyges
*(treat the 1–3 height as a value-label and it's Tier-0; otherwise nudge toward the
stacking primitive)*.

**Hopper / custom-adjacency games (square + value/label, custom move-gen)** —
*critic fixes, promoted from "blocked":*
- **Xiangqi (Chinese Chess)** — 9×10 grid; cannon = a hopper (custom Python), river
  + palace = positional rules. Pieces-on-intersections render fine as pieces-in-
  cells; the drawn lines are cosmetic-only. One of the most-played games on Earth.
- **Janggi (Korean Chess)** — Xiangqi sibling; cannon hopper + palace diagonals.
- **Fanorona** — alquerque adjacency (some intersections diagonal) in code +
  approach/withdrawal capture; lines cosmetic.
  *(These build a reusable **hopper/cannon core** — see engine cores below.)*

**Go cousins WITHOUT territory scoring (square)** — **Atari/Capture Go**, **NoGo**,
**Gonnect**, **Tanbo**. Need only liberty/group-capture + ko repetition — *not*
scoring/dead-stones/two-pass. These ship today and **de-risk the eventual Go
investment** (the liberty engine is the shared part).

**Group-by-event games (square/hex)** — Catchup/Ketchup, Symple, Storisende,
Starweb, Meridians. Win by comparing own-group sizes (no empty-region counting).
Oust/Yodd are the reference fits.

**Value-on-cell (square, text-label rendering)** — **Cephalopod** (deterministic
placed "dice" = numeric tokens, capture-merge), **Mancala family** *(critic add:
Oware/Kalah/Bao — per-pit seed counts render as number labels; sowing is custom
logic; pits + 2 stores as a polygons/square layout. The oldest game family on
Earth, entirely missed by the first pass)*.

## Reusable engine cores (code only — NO UI/renderer change)

Self-contained engine work that each unlocks a cluster; all perft/known-result
checkable. Not "platform gaps" — they're libraries like `chesslike.py`.

| Core | Effort | Unlocks |
|---|---|---|
| **Hex ChessLike** (6 rook + 6 bishop dirs, hex-knight leaps, hex-pawn geometry) | M | Glinski's, McCooey's, Shafran's Hex Chess; Hexdame. Reuses the existing hex board. |
| **Cannon / hopper core** (screen-jump move-gen) | S–M | Xiangqi, Janggi, and any chessvariant hopper. |
| **Go liberty/capture core** (groups, liberties, ko/superko) | M | Atari Go, NoGo, Gonnect, Tanbo now; the base layer of full Go later. |

## Platform / UI investments (ranked by fame-per-effort)

These are the *genuine* renderer/engine-UI primitives we lack.

### 1. Hand / reserve / pocket — drop pool · **effort M** · **highest leverage**
Per-seat off-board reserve in state + a drop move syntax (`*P@c,r`) + a small
reserve tray in the UI. Unlocks the **whole Morris family** (Nine/Twelve/Three
Men's Morris, Morabaraba, Tapatan — Nine Men's Morris is a solved classic),
**Crazyhouse** (the cleanest drop test, on the existing chess board), **Shogi**
*(critic: deserves first-class status, not a footnote — drops + promotion zone)*,
Tori Shogi, Bagh-Chal (Tigers & Goats), Mijnlieff. Mostly state + one move type +
a tray; no new board renderer. *(Morris adjacency can be faked on square/polygons
with custom adjacency; the mill-removes-a-stone is custom logic.)*

### 2. Stacking — ordered owned towers · **effort L**
Ordered multi-piece cell state + stack/spread move syntax + a renderer showing
height/top-owner (count badge or layered discs) + MCTS awareness. Biggest unlock
of famous *modern* abstracts: **Tak** (flagship, road-building), **DVONN**,
**TZAAR**, **Focus/Domination**, **Lasca**, **Emergo**, Battle Sheep
*(hex/polygons, split-a-stack-and-move)*, **Gobblet** *(critic add: nesting)*,
**Pylos** *(critic add: 3-D pyramid of balls)*. Touches state, move grammar,
renderer, and bot → L.

### 3. Territory / area scoring — **effort L** · high prestige, low breadth
Region-flood scoring, dead-stone marking UI, two-pass termination, ko/superko.
Unlocks the **flagship hole — Go (9/13/19)** and little else (Sygo, Spargo). Ranked
behind drops/stacking *because* the light Go cousins already ship in Tier 0 and the
liberty core (above) is the shared half — so this is "the scoring/UI layer on top."

### 4. Point-and-line board — drawn edges as first-class state · **effort L**
Edges that are placed and mutually block / define capture paths (distinct from
filled polygons). Unlocks **TwixT** (knight's-move bridges that block), **Surakarta**
(loop-arc capture tracks), the **GIPF intersection boards** (GIPF/YINSH/PÜNCT, which
also need stacking/flipping), Sprouts. Smaller marginal unlock since several targets
*also* need other primitives. (Xiangqi/Janggi/Fanorona/Morris do **not** need this —
custom adjacency on existing boards suffices, see Tier 0.)

### 5. Multi-action turn grammar generalization · **effort S**
A move-string convention + `apply_move` work (Amazons' `move>shoot` already proves
two-segment moves). Unlocks Pentago (place + rotate a quadrant), Slither (place +
slide), Quoridor (move **or** place a wall — also a light edge/wall touch), Arimaa
(up to 4 steps, push/pull), Connect6. Cheapest, but each game still carries custom
logic so per-game breadth is modest.

### 6. >2-player seating UI · **effort M**
Engine already accepts any `num_players`; the web UI is 2-seat. Adds seat-color
routing, turn cycling, >2 result handling. Pure-UI unlocks: Chinese Checkers
(Sternhalma), Halma (4p), 4p Quoridor. (Blokus/Battle Sheep also need other
primitives.)

### 7. Dice / card UI · **effort M** · lowest priority
Engine has CHANCE nodes but bot + UI are immature. Onitama (open-info shared
movement cards — most tractable), EinStein würfelt nicht, Entropy/Hyle. Small
unlock set; MCTS needs chance-node hardening.

### Cheap renderer niceties (opportunistic, ~S each)
- **Per-cell arbitrary color** → **Kamisado** (color-routing movement) + nicer
  boards generally.
- **Height badge / two-things-per-cell** → **Santorini** *(critic fix: its "stack"
  is per-cell terrain height 0–3 + dome + one worker — a value-on-cell + piece
  overlay, NOT owned towers; much closer to Tier 0 than to the stacking primitive)*.
- **Neutral pieces with attributes** → **Quarto** *(critic add: shared pieces with
  4 binary attributes + the "opponent picks your piece" turn grammar)*.

## Suggested first fan-out batch (Tier 0, all anchorable)

A high-value, low-risk opening salvo to validate the mass-production workflow:
**Tablut + Hnefatafl** (tafl sizes), **International Draughts + Konane**,
**Havannah + Y**, **Makruk + Shatranj + King-of-the-Hill** (ChessLike pack),
**Atari Go + NoGo** (exercise the new liberty core), and **Xiangqi** (exercise the
new hopper core). Each ships with a `rules.md`, a conformance pass, and an
independent correctness anchor or it's flagged for review — never auto-merged.
